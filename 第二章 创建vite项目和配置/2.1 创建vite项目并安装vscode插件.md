# 2.1 创建vite项目并安装vscode插件

## 1. 访问v3官网

https://cn.vuejs.org/



## 2.访问vite官网

https://cn.vitejs.dev/

 

## 3. 查看npm版本

`npm -v`



## 4. 查看npm镜像

`npm config get registry`



## 5. 设置为npm源为国内镜像

`npm config set registry=国内镜像地址`



## 6. 使用vite命令创建项目

`npm create vite@latest shop-admin -- --template vue`



## 7. 切换到项目目录

`cd shop-admin`

## 8. 安装依赖

`npm install`

## 9. 启动项目

`npm run dev`
